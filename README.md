# Extrator de Excel

## Visão Geral

Este projeto é um aplicativo web que permite aos usuários extrair dados de arquivos Excel e apresentá-los de forma organizada. O extractor foi desenvolvido usando HTML, CSS e JavaScript, proporcionando uma interface interativa e amigável.

## Tecnologias Utilizadas

- **HTML:** Para a estruturação da página e dos elementos do formulário.
- **CSS:** Para estilizar a aplicação, tornando-a visualmente atraente e responsiva.
- **JavaScript:** Para implementar a lógica de extração de dados e manipulação de arquivos.

## Funcionalidades

- Upload de arquivos Excel (.xls, .xlsx).
- Visualização dos dados extraídos em tempo real.
- Possibilidade de filtrar e buscar dados específicos.
- Download dos dados extraídos em diferentes formatos (CSV, JSON).

## Como Usar

1. Acesse a página do aplicativo.
2. Clique no botão "Selecionar Arquivo" e escolha o arquivo Excel que deseja extrair.
3. Após o upload, os dados serão exibidos na tela.
4. Utilize as ferramentas de filtragem para encontrar os dados desejados.
5. Baixe os dados extraídos se necessário.
